# HiRiQ-solver
Java HiRiQ Puzzle (also called Peg Solitaire) solver using implementation of Trees, Queues and BFS (Breadth First Search)

### How to run
```
javac pegSolitaireSolver.java
java pegSolitaireSolver
```
